## 1. 产品概述

福尔摩斯式推理系统——案件卡片系统，是一个将侦探推理过程可视化的交互式Web应用。用户通过浏览案件卡片、展开线索细节、手动选择嫌疑人来完成推理，系统不会自动给出答案，而是让用户自行体验推理的乐趣。

- 目标用户：推理爱好者、解谜游戏玩家、逻辑思维训练者
- 核心价值：将抽象的推理过程转化为可视化的线索探索与嫌疑人筛选交互

## 2. 核心功能

### 2.1 用户角色

无需角色区分，所有用户均可直接使用。

### 2.2 功能模块

1. **案件大厅页面**：案件卡片列表、氛围感背景、案件筛选
2. **案件详情页面**：线索展示与展开、嫌疑人选择、推理记录面板

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 案件大厅 | 案件卡片列表 | 展示所有案件卡片，每张卡片包含标题、时间、地点摘要 |
| 案件大厅 | 氛围头部 | 维多利亚时代风格的标题区，营造福尔摩斯氛围 |
| 案件详情 | 案件信息栏 | 展示案件标题、时间、地点的完整信息 |
| 案件详情 | 线索面板 | 三条线索，点击展开查看线索细节，支持收起 |
| 案件详情 | 嫌疑人选择区 | 展示嫌疑人列表，用户可手动选择一位嫌疑人 |
| 案件详情 | 推理记录面板 | 记录用户的线索查看和嫌疑人选择操作 |

## 3. 核心流程

用户进入案件大厅，浏览案件卡片列表，点击感兴趣的案件进入详情页。在详情页中，用户逐一点击线索展开查看细节，根据线索信息在嫌疑人列表中选择自己认为的犯人。整个推理过程在推理记录面板中可视化呈现。

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深棕褐 (#2C1810) + 羊皮纸黄 (#D4B896) + 暗红点缀 (#8B2500)
- 辅助色：墨绿 (#2D4A3E)、烟灰 (#6B6B6B)
- 按钮风格：仿古皮质按钮，圆角微浮雕效果，悬停时轻微发光
- 字体：标题使用衬线体（Playfair Display），正文使用清晰无衬线体（Crimson Text）
- 布局：卡片式布局，模拟案卷档案的视觉感
- 图标风格：手绘线条风格，类似维多利亚时代墨水画

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 案件大厅 | 氛围头部 | 深色背景、雾气效果、标题使用Playfair Display衬线体、放大镜图标装饰 |
| 案件大厅 | 案件卡片列表 | 羊皮纸质感卡片、3列网格布局、卡片悬停时微微倾斜上浮、卡片边框仿旧纸张效果 |
| 案件详情 | 案件信息栏 | 顶部横幅区域、暗红分割线、信息以标签形式展示 |
| 案件详情 | 线索面板 | 信封式展开动画、线索编号标记、展开时纸张翻转效果、线索文字使用打字机字体 |
| 案件详情 | 嫌疑人选择区 | 人物头像占位、悬停高亮、选中后红色印章标记效果 |
| 案件详情 | 推理记录面板 | 侧边时间线、墨水笔迹风格记录条目 |

### 4.3 响应式设计

- 桌面优先设计，3列卡片网格
- 平板适配为2列
- 移动端单列布局，线索面板全宽展示

### 4.4 动效设计

- 页面加载：卡片依次淡入，模拟翻阅案卷
- 线索展开：信封开启式动画，内容从折叠到展开
- 嫌疑人选中：印章盖下效果，伴随轻微震动
- 页面切换：烟雾消散式过渡
